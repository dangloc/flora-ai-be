🔧 CUSTOMIZATION GUIDE
═════════════════════════════════════════════════════════════════════════════

To change the color scheme, edit chatBubble.css:

1. Find all instances of:
   linear-gradient(135deg, #667eea 0%, #764ba2 100%)

2. Replace with your gradient colors:
   linear-gradient(135deg, YOUR_COLOR1 0%, YOUR_COLOR2 100%)

Examples:
  • Blue:   #1890ff → #0050b3 (Ant Design default)
  • Green:  #52c41a → #389e0d
  • Pink:   #eb2f96 → #c41d7f
  • Red:    #f5222d → #cf1322
