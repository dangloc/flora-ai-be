## 🚀 How to Access

1. **Login as Admin**: http://localhost:3000/login
2. **Go to Chat Support**: http://localhost:3000/admin/chat-support
3. **See sidebar with user conversations**
4. **Click user to open chat**
5. **Send/receive messages in realtime**
