HOW IT WORKS
============

User sees website
         ↓
Chat button appears (bottom-right)
         ↓
User clicks button
         ↓
Modal opens with chat
         ↓
User types message
         ↓
Message sent to GlobalState chatAPI
         ↓
AI responds
         ↓
Response displayed in modal
         ↓
User can continue chatting or close


CUSTOMIZATION EXAMPLES
======================

Change button color:
  Edit chatBubble.css → .chat-bubble-button
  Change background gradient colors

Change modal width:
  Edit ChatBubble.js → Modal component
  Change width={400} to width={500}

Change header text:
  Edit ChatBubble.js → Modal title prop
  Replace "Tư vấn sản phẩm thời trang"

Change message bubble colors:
  Edit chatBubble.css → .message-bubble
  Modify background colors
